Chapter 9 - File I / O

  
                The random acess memory is volatile and all its conents are last once a program terminates in order to persist the data forever , 
                 we use files . A file is data stored in a storage device . A puthon program can talk to hthe file by reading content form it
                 and writing content to it
                    
                    
 Types of file  

                    
                        There are two types of files:
                         1. Text files (.txt, .c etc)
                         2. Binary files (.jpg,.dat,etc)
            
                         python has a lot of functions for reading, updating and delefing files
                  
                    
 Opening a file 

                    
                    Python has an open () function for opening files. It takes 2 parameters: Filename and mode 
                       open ( "this .txt","r")
                       here "r" => mode of opening (read mode)
                            " this .txt" => Filename
                            open => open is built in function  
                    
 Reading a file in python 

                        f = open( " this.txt","r") ---> open the file in r mode 
                        text = f.read() ----> Read its Contents
                        print(text)---> print its Contents
                        f.close()---> close the file 
            
                    we can also specify the number of charecter in read function : f.read(2) ---> Read first two charecter
                    
 Other methods to read 

                    We can also use f.readline() function to read o  full line at a time
                          f.readline()  ----> Reads one line from the file 
                    
 Modes of opening a file

                    r ----> open for reading
                    w -----> open for writing
                    a -----> open for appending
                    + -----> open for updating
                    
                    'rb' will open for read in binary mode 
                    'rt' will open for read in text mode
                    
 Writing files in python

                    In  order to write to a file , we first open it in write or append mode after which , we can the python's 
                    f.write() method to write to the file ! 
                     f= open("this.txt","w")
                     f.write ("This is nice") ---> can be called multiple times 
                     f.close()
            
                    
 With statement

                    The best way to open and close the file automatically is the with statement 
                       with open("this.txt") as f:
                       f.read() ----> don't need to write f.close() as it done automatically
