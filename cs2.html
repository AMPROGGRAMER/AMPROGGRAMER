 What is DOM?

 DOM stands for document object model When a page is loaded, the browser creates a DOM of the page which is constructed as a tree of Objects!
            
 HTML id and class attributes
When an HTML element is given an id, it serves as a unique identifier for that element On the other hand, when an HTML element is given a class, it now belongs to that class. More than one elements con belong to a single class but every element must have a unique id (if assigned).
We can add multiple classes to an element like this
 Unique id
< div id=”first” class = “C1 C2 C3 “>
                              ↳ multiple classes followed by Spaces
