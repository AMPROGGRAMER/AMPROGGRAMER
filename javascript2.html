 Conditional Statements

            Sometimes we might have to execute a block of code based of some condition. 
            for example "a prompt might ask for the age of the user and if its greater than 18, display a special message. 
            In Javascript we have three forms of if ... else Statement;
            1.  if statement
            2. if... else statement
            3. if ... else if ... else statement
            

                if  stalement 

            The if statement in JavaScript looks like this:
                              if (condition) { 
                                           // execute this code 
                                             }
            The if statement evaluates the condition inside the (). if the condition is evaluated to true, the code inside the body of if is not executed else the code is not executed. 
            
         
   if-else Statement

            The if statement can have an optional else clause. The syntax looks something like this
                           if (condition) {
                                      // block of code if condition thue
                                          }
                                      else {
                                     //block of code if condition false
                                           }
            If the condition is true, code inside if is executed else code inside else block is executed.
