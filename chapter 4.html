CHAPTER 4 – LIST AND TUPLES

 
        Python list are containers to store a set of value of  any type of data
        Ex.    Fruits = [ “apple”,”Akash”,”Rohan”,7,”False”]
                (Can store  value of any datatype)

List /Indexing 

 
            A list can be indexed just like a string 
            L1 = [7,9,”Herry”]
            L1[0] = 7
            L1[1] = 9
            L1[70] = error
            L1[0,1] = [7,9]

List Method 

 
            Consider the following list :
            L1 = [ 1,8,7,2,21,15]
1]	L1.sort() : Update the list to [1,8,7,2,21,15]
2]	L1.reverse() : Update the list to [15,21,2,7,8,1]
3]	L1.append(8) : Add 8 at the end of list 
4]	L1.insert(3,8) : This will 8 at 3 index
5]	L1.pop(2) : Will delete element at index 2 and return it’s value
6]	L1.remove(21) : Will remove 21 from list

Tuples in python

 
            Tuple is an immutable (cannot change )data type in python 
            a = ()    =>  Empty tuple 
            a = (1)   =>.   tuple with only one element needs a comma 
            a = (1,7,2)   =>    tuple with more than 1 element 
          Once defined a tuples element cannot be altered or manipulated
